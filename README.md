# AI Cover Letter Generator

This project generates personalized cover letters using AI by analyzing a resume and job description.

## Features
- Resume text extraction
- Job description analysis
- AI-generated cover letters
- Supports multiple file formats

## Installation

Install dependencies

pip install -r requirements.txt

Run the application

python app.py

## Technologies Used
- Python
- OCR
- NLP
- Local LLM
